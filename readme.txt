Interaktive Gebäudekarte als WordPress-Plugin. Stockwerke und Kartenelemente werden in der WordPress-Datenbank gespeichert.

== Description ==

WP Floormap ermöglicht es Ihnen, interaktive Gebäudekarten direkt in WordPress zu verwalten und anzuzeigen. Sie können Stockwerke mit eigenen Kartenbildern anlegen, Kartenelemente wie Polygone oder Icons platzieren und diese mit detaillierten Tooltips und Info-Sheets versehen.

Die Karte basiert auf der Leaflet-Bibliothek und ist für Performance optimiert (u.a. durch automatische WebP-Konvertierung).

Die Einbindung erfolgt flexibel per Shortcode, Widget oder Elementor-Widget.

== Installation ==

1. Laden Sie den Plugin-Ordner in das Verzeichnis `/wp-content/plugins/` hoch.
2. Aktivieren Sie das Plugin im Menü 'Plugins' in WordPress.
3. Konfigurieren Sie Ihre Stockwerke im Admin-Bereich unter 'WP Floormap'.
4. Nutzen Sie den Shortcode `[wp_floormap]` auf Ihren Seiten.

== Frequently Asked Questions ==

= Wie binde ich die Karte ein? =
Nutzen Sie den Shortcode `[wp_floormap]` auf einer beliebigen Seite oder verwenden Sie das Elementor-Widget.

= Welche Bildformate werden unterstützt? =
Standardformate wie JPG und PNG werden unterstützt und beim Upload automatisch in das performante WebP-Format konvertiert.
