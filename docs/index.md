# Willkommen zu WP Floormap

Interaktive Gebäudekarte als WordPress-Plugin.

## Features
- **Stockwerke verwalten**: Kürzel, Name, Kartenbild einfach im Backend pflegen.
- **Optimierte Performance**: Automatische Konvertierung nach WebP und Skalierung von Rasterbildern.
- **Interaktive Elemente**: Polygone und Icons mit Tooltips und Info-Sheets (Leaflet-basiert).
- **SVG-Support**: Automatische Bereinigung und Optimierung von SVGs für beste Qualität.
- **Elementor-Integration**: Eigenes Widget für den Elementor Page Builder.
