# Installation

Befolgen Sie diese einfachen Schritte, um WP Floormap zu installieren:

1. **Plugin-Ordner hochladen**: Kopieren Sie den Plugin-Ordner in das Verzeichnis `wp-content/plugins` Ihrer WordPress-Installation.
2. **Aktivieren**: Aktivieren Sie das Plugin im WordPress-Backend unter "Plugins".
3. **Datenbank**: Die benötigten Datenbanktabellen werden automatisch beim Aktivieren erstellt.
4. **Verzeichnisse**: Das Upload-Verzeichnis `wp-content/uploads/wp-floormap` wird ebenfalls automatisch angelegt.

Sobald installiert, können Sie mit der [Verwendung]({{ site.baseurl }}/usage/) fortfahren.
